## Tech & Build Notes

- Fully self-contained single-file HTML (`index.html`) — no build step required.
- Canonical role data lives in `data/roles.json` (easy to keep README table + UI in sync).
- Interactive features: category filters, keyboard shortcuts (`/` resets, `?` shows top-comp roles), Web Share API + print-optimized poster view.
- Styling: Tailwind via CDN + custom glassmorphism + Font Awesome 6.
